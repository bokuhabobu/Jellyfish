# Healing Jellyfish — 3D Visual Project

地域共創 3Dbiz研究会で制作した、Looking Glass向けの立体映像作品です。

Blenderでクラゲをモデリングし、傘の収縮や触手の揺れを調整しながら、深海をゆっくり漂うクラゲの映像を制作しました。静止画ではなく、動いたときにもクラゲらしく見えることを意識したチーム制作です。

## このサイトに載せている内容

- 作品の目的と担当した制作内容
- チームで方向を見直しながら進めた制作の歩み
- Blenderでのモデリング、アニメーション、レンダリングの考え方
- モーションテストからLooking Glassでの立体視確認までの映像
- 制作を通して学んだ、触手表現と動きの難しさ


## 使用ツール

- Blender
- Looking Glass
- HTML / CSS / JavaScript
- GitHub Pages
